# grid.test.ts

## 責務

格子ヘルパーのキー、等価性、整数制約、world 変換、隣接方向を固定する。

## 関数仕様

### void describe('grid helpers')

`{2,-3}` のキーが `2:-3` で安定すること、`{1,2}` 同士は同じで `{2,1}` とは違うこと、整数 `{0,-1}` のみ有効で x=.5/z=-1.5 は無効、`{3,-2}`, cell4,y1 が `{12,1,-8}` になることを検証する。origin `{4,5}` の north=`{4,4}`、east=`{5,5}`、`{4,6}` の方向south、2セル東はnull。
