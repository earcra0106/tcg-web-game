# DevContainer Overview

## 概要

`echo $REMOTE_CONTAINERS` の出力が `true` の場合、この環境はdevcontainerです。

## コマンド実行について

devcontainer 内には node がインストールされています。
パッケージマネージャーは pnpm です。
設計の際、実装予定の機能の関連をwebで調査し、既存のパッケージを使って解決できるかどうかを確認してください。解決可能なパッケージが場合は、そのパッケージを導入することを提案してください。
解決可能なパッケージがあるにもかかわらずコードを自作することはなるべく避け、すべて自作が必要な要件とパッケージで解決できる要件は、分けて設計書に記載して下さい。

## pnpm 運用について
古いバージョンがインストールされるのを防ぐため、新しいパッケージをインストールする際は `pnpm add` を使用してください。
`pnpm install` は、パッケージが解決できないなどの不具合の際に実行してください。

```bash
cd /workspaces/apps && pnpm add <package-name>
```
