# 操作仕様

## 概要

PC とスマートフォンの両方で、グリッド上にマシンを配置し、マシン同士をコンベアで接続できる操作体系にします。

マシン自体は向きを持ちません。素材の搬送方向は、コンベア接続によって決まります。

## PC 操作

### カメラ操作

| 操作 | 挙動 |
| --- | --- |
| 右ドラッグ | カメラをパンします。 |
| マウスホイール | ズームイン、ズームアウトします。 |

カメラ操作は、左クリックによる配置や選択と競合しないようにします。

### マシン配置

PC では、以下の2種類の配置操作に対応します。

| 操作 | 挙動 |
| --- | --- |
| ツールバーからマシンをドラッグ | ドロップしたグリッドセルにマシンを配置します。 |
| ツールバーでマシンをクリックして選択し、グリッドセルをクリック | 選択中のマシンをクリックしたセルに配置します。 |

配置物は整数座標のグリッドにスナップします。既存の配置物があるセルには重複配置できません。

### コンベア接続

コンベアはツールバーで選択したあと、マシン間を接続して作成します。

| 操作 | 挙動 |
| --- | --- |
| 接続元マシンをクリックし、接続先マシンをクリック | 接続元から接続先へ向かうコンベアを作成します。 |
| 接続元マシンで押し続け、接続先マシン上で離す | ドラッグ操作として同じ接続を作成します。 |

生成されたコンベアは、接続元から接続先へ向かう方向付き搬送路として扱います。

### 削除

ツールバーの削除ツールを選択し、対象のマシンまたはコンベアをクリックすると削除します。

## スマートフォン操作

### カメラ操作

スマートフォンではタッチ操作でカメラを操作します。

| 操作 | 挙動 |
| --- | --- |
| 1本指ドラッグ | 盤面をパンします。 |
| 2本指ピンチ | ズームイン、ズームアウトします。 |

### マシン配置

下部ツールバーでマシンを選択し、配置したいグリッドセルをタップして配置します。

PC と同様に、マシンは整数座標のグリッドにスナップします。既存の配置物があるセルには重複配置できません。

### コンベア接続

コンベアを選択し、接続元マシン、接続先マシンの順にタップして接続します。

スマートフォンでは、誤操作を避けるため、クリック選択方式を基本とします。

### 削除

削除ツールを選択し、対象のマシンまたはコンベアをタップして削除します。

## 共通操作

配置済みマシンを選択すると、そのマシンの状態とレシピ指定を確認できます。

マシンにレシピが指定されている場合は、そのレシピを優先して処理します。指定されていない場合は、入力された素材から作成可能なレシピを自動判定します。
