# ゲーム進行仕様

## 概要

Voxel Kitchen Automation は、料理のお題に合わせて調理ラインを構築し、必要効率を満たすことを目指すステージ制の自動化シミュレーションゲームです。

明確なゲームオーバーは設けず、プレイヤーは条件達成までラインを改善し続けられます。

## 基本ループ

1. ステージのお題を確認する。
2. 必要な料理、素材、要求効率を把握する。
3. マシンとコンベアを配置し、調理ラインを構築する。
4. ラインを稼働させ、食材を搬送・加工・出荷する。
5. 稼働結果を見ながら、配置や接続を改善する。
6. 指定料理の出荷効率が要求値を満たしたらステージクリアとする。
7. 次のステージへ進む。

## ステージ進行

ステージは、同じシード値とステージ番号で同じお題が生成される構成にします。

お題には、対象レシピと必要効率を含めます。必要効率は、一定期間あたりに出荷口へ届けられた指定料理の数で判定します。

## 編集と稼働

プレイヤーは稼働中もマシン配置、削除、レシピ指定、コンベア接続を編集できます。

編集によって一時的にラインが詰まった場合でも失敗扱いにはせず、素材やマシンは仕様に従って待機します。プレイヤーは状況を確認しながら、より効率のよいラインへ改善します。

## クリア条件

ステージクリア条件は、指定された料理を出荷口へ届け、必要効率を満たすことです。

累計納品数だけではクリアにせず、安定した自動化ラインを作ることを重視します。クリア時には成功演出と進行状態の更新を行います。

## 導入ステージ

最初のステージ群では、以下の要素を順番に扱います。

1. 倉庫から基本素材を出す。
2. コンベアでマシン同士を接続する。
3. 単一素材を切断または加熱する。
4. 複数素材を待機させて合体またはミキサー処理する。
5. 完成料理を出荷口へ届け、必要効率を満たす。

チュートリアル専用モードは設けず、初期ステージの中で自然に操作と流れを覚える構成にします。
