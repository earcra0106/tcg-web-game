# マシン仕様

## 概要

マシンはグリッド上に配置される調理設備です。マシン自体は向きを持たず、素材の流れはコンベア接続によって決まります。

## 共通仕様

| 項目 | 仕様 |
| --- | --- |
| 配置 | 整数座標のグリッドセルに配置します。 |
| 重複配置 | 既存の配置物があるセルには配置できません。 |
| 向き | マシン自体は向きを持ちません。 |
| 入力 | コンベア接続から素材を受け取ります。 |
| 出力 | 接続された次のマシンまたは出荷口へ素材を渡します。 |
| 処理時間 | マシン種別ごとの固定値を持ちます。 |
| 出力詰まり | 出力先が満杯または未接続の場合、素材はその場で待機します。 |

## レシピ指定

配置済みマシンをクリックまたはタップすると、レシピを指定できます。

指定されたレシピがある場合、マシンはそのレシピを優先して処理します。レシピが指定されていない場合は、入力された素材から作成可能なレシピを自動判定します。

入力素材がレシピ条件を満たさない場合、素材はマシン内に保持されます。複数素材が必要なマシンでは、必要素材がそろうまで内部で待機します。

## コンベア

コンベアは、マシン間を接続する方向付き搬送路です。

プレイヤーは接続元マシンと接続先マシンを指定します。接続操作によって、接続元から接続先へ向かうコンベアが生成されます。

コンベアは素材を1マスずつ搬送します。次の受け渡し先が受け取り可能であれば素材を渡し、受け取りできない場合はコンベア上で待機します。

## マシン一覧

| マシン | 役割 | 操作・挙動 |
| --- | --- | --- |
| コンベア | 素材を移動する | 接続元から接続先へ向かう搬送路として生成されます。 |
| 分岐機 | ラインを複数方向へ分配する | 入力された素材を分配ルールに従って複数の接続先へ振り分けます。 |
| 合流機 | 複数入力を1出力へ統合する | 複数ラインから来た素材を1本の出力ラインにまとめます。 |
| 切断機 | 食材を刻む、切る、スライスする | 1つの素材を、指定または自動判定された切断レシピの結果へ変換します。 |
| 加熱機 | 食材を焼く、煮る、炊く | 1つ以上の素材を、指定または自動判定された加熱レシピの結果へ変換します。 |
| ミキサー | 素材を混ぜる | 素材の形が残らない料理や中間素材へ変換します。 |
| 合体機 | 素材を盛り付ける | 素材の形が残る組み合わせ料理へ変換します。 |
| ゴミ箱 | 不要物を除去する | 入力された素材を消去し、ラインを整理します。 |

## 分岐と合流

分岐機は、複数の出力接続を持てます。分配ルールは、素材種別または順番による振り分けを想定します。

合流機は、複数の入力接続を受け取れます。出力先が受け取り可能になるまで素材を保持し、ラインの詰まりを防ぎます。

## 待機と詰まり

素材は未接続や満杯を理由に自動破棄されません。

出力先がない場合、または出力先が受け取りできない場合は、素材を保持して待機します。プレイヤーは削除ツール、接続変更、レシピ指定により詰まりを解消します。
