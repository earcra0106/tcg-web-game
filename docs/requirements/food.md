# たべもの要件定義

## 1. 概要

調理ラインで扱う食材、中間素材、完成料理、調理工程を定義する。

完成料理の構成は、以下の方針に従う。

- 「BLTサンド」は日本では一般的ではないため採用しない
- 完成料理に「ごはん」「チャーハン」を含める
- 基本食材は、完成料理の依存素材として 3 種類以上の料理で使われるようにする
- 「ごはん」と「オムレツ」は、完成料理としても指定され、ほかの完成料理の素材としても使われる
- 料理ごとのスコア、必要効率、アンロック順の具体値は、この要件定義には記載しない

## 3. 調理工程・マシン

| マシン | 役割 | 仕様 |
| --- | --- | --- |
| コンベア | 素材を移動する | 1 入力 1 出力で、向きに沿って次のマスへ送る |
| 分岐機 | ラインを複数方向へ分配する | 入力された素材をルールに従って複数方向へ振り分ける |
| 合流機 | 複数入力を 1 出力へ統合する | 複数ラインから来た素材を 1 本のラインにまとめる |
| 切断機 | 食材を刻む、切る、スライスする | 1 つの素材を別の単一素材へ変換する |
| 加熱機 | 食材を焼く、煮る、炊く | 1 つ以上の素材を加熱済みの素材または料理へ変換する |
| ミキサー | 素材を混ぜる | 素材の形が残らない加工に使う。ソース、ポテトサラダ、ハンバーグのたね、チャーハンなどが対象 |
| 合体機 | 素材を盛り付ける | 素材の形が残る組み合わせに使う。サンドイッチ、サラダ、カレーライス、グラタンの組み立てなどが対象 |
| ゴミ箱 | 不要物を除去する | 不要な素材を消し、ラインを整理する |
